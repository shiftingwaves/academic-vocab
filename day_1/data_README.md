### 2. EXPLANATION

*   **Data:** Information, facts, or numbers collected for study.
    *   *Roman Urdu:* Wo maloomat ya numbers jo hum kisi research ya computer mein jama karte hain.
*   **Database:** A structured set of data held in a computer.
    *   *Roman Urdu:* Computer mein wo jagah jahan bohot sara data organize karke rakha jata hai.
*   **Dataset:** A collection of related data.
    *   *Roman Urdu:* Ek jaisa data ka group jo kisi khaas maqsad ke liye jama kiya gaya ho.
*   **Data-driven:** Controlled or guided by data.
    *   *Roman Urdu:* Jab koi faisla sirf data ko dekh kar kiya jaye, na ke andazon par.

---

### 3. SYNONYMS

*   **Data:**
    *   **Information:** Maloomat.
    *   **Statistics:** Shumariyat (numbers/facts).
    *   **Facts:** Haqayiq.

---

### 4. ACADEMIC EXAMPLES

*   **Data:**
    1. The researcher collected **data** from 500 users. (Researcher ne 500 users se data jama kiya.)
    2. We need more **data** to prove this theory. (Is theory ko prove karne ke liye humein mazeed data chahiye.)
*   **Database:**
    1. The **database** stores all student records. (Database mein tamam students ka record mehfooz hai.)
    2. SQL is used to manage the **database**. (SQL ka istemal database ko manage karne ke liye hota hai.)
*   **Dataset:**
    1. The AI model was trained on a large **dataset**. (AI model ko ek bare dataset par train kiya gaya.)
    2. This **dataset** contains errors. (Is dataset mein galtiyan hain.)
*   **Data-driven:**
    1. This is a **data-driven** company. (Ye ek data-driven company hai.)
    2. We made a **data-driven** decision. (Humne data ko dekh kar faisla kiya.)

### 5. MIXED ACADEMIC SENTENCES

1. The experiment provided reliable **data**. (Experiment ne qabil-e-aitimad data diya.)
2. Access the **database** to find the results. (Results dhoondne ke liye database access karein.)
3. We analyzed the **dataset** carefully. (Humne dataset ko gaur se analyze kiya.)
4. Modern software is highly **data-driven**. (Jadeed software bohot zyada data-driven hote hain.)
5. Please save the file in the **database**. (Baraye meharbani file ko database mein save karein.)
6. The **data** shows a clear trend. (Data ek saaf trend dikha raha hai.)
7. Researchers often clean their **dataset** first. (Researchers aksar pehle apna dataset saaf karte hain.)
8. This **data-driven** approach is very effective. (Ye data-driven approach bohot asar-dar hai.)
9. The **database** crashed due to too much traffic. (Zyada traffic ki wajah se database crash ho gaya.)
10. We found interesting patterns in the **data**. (Humne data mein dilchasp patterns paye.)
11. Is this **dataset** publicly available? (Kya ye dataset aam logon ke liye available hai?)
12. The system uses a **data-driven** algorithm. (System ek data-driven algorithm istemal karta hai.)
13. Update the **database** with new entries. (Naye entries ke saath database ko update karein.)
14. The **data** supports our hypothesis. (Data hamari hypothesis ko support karta hai.)
15. A good **dataset** is key to success. (Ek achha dataset kamyabi ki kunji hai.)

**Memory Trick:**
*   **Data** is the "Food" for computers.
*   **Database** is the "Fridge" where you keep the food.
*   **Dataset** is a "Meal" (a specific set of food) prepared for a specific task.
