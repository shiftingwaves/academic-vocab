### 2. EXPLANATION

*   **Constitute (v):** To be the parts that make up a whole. (Kisi cheez ka hissa hona ya mil kar banna.)
*   **Constituted (v/adj):** Made up of. (Jo cheezon se mil kar bani ho.)
*   **Constitutes (v):** Forms or makes. (Ye word singular subject ke sath use hota hai.)
*   **Constituting (v):** The act of forming something. (Kisi cheez ko banane ka amal.)
*   **Constitution (n):** The basic set of rules/laws. (Qanoon-e-asas ya bunyadi usool.)
*   **Constitutions (n):** Plural of constitution. (Ek se zyada qanoon-e-asas.)
*   **Constituent (n/adj):** A part of a whole. (Kisi bade system ka ek chota hissa.)
*   **Constituents (n):** Plural of constituent. (Bohat sare hissay.)
*   **Constituency (n):** An area or group represented by someone. (Wo area jahan se log elect hote hain.)
*   **Constituencies (n):** Plural of constituency. (Bohat sare areas.)
*   **Constitutional (adj):** Allowed by the law. (Jo qanoon ke mutabiq ho.)
*   **Constitutive (adj):** Essential part of something. (Zaroori hissa hona.)
*   **Unconstitutional (adj):** Against the law. (Jo qanoon ke khilaf ho.)
*   **Constitutionally (adv):** In a way that follows the law. (Qanooni taur par.)

---

### 3. SYNONYMS

*   **Constitute:** Make up (banana), Form (tashkeel dena), Compose (mil kar banna).
*   **Constitution:** Law (qanoon), Charter (dastoor).
*   **Constituent:** Component (hissa), Element (juz).

### 4. ACADEMIC EXAMPLES

1.  **Constitute:** Data points constitute the foundation of AI models. (Data points AI models ki bunyad banate hain.)
2.  **Constituted:** The software is constituted of several modules. (Ye software bohat sare modules se mil kar bana hai.)
3.  **Constitutes:** Coding constitutes a major part of this course. (Coding is course ka ek bada hissa hai.)
4.  **Constituting:** Constituting a secure network is our priority. (Ek secure network banana hamari priority hai.)
5.  **Constitution:** The constitution of the system defines user access. (System ka dastoor user access ko define karta hai.)
6.  **Constituent:** Hardware is a key constituent of a computer. (Hardware computer ka ek ahem hissa hai.)
7.  **Constituency:** The university constituency voted for the new lab. (University ke area ne new lab ke liye vote diya.)
8.  **Constitutional:** This policy change is constitutional. (Ye policy change qanoon ke mutabiq hai.)
9.  **Constitutive:** These rules are constitutive of our research. (Ye usool hamari research ka zaroori hissa hain.)
10. **Unconstitutional:** Hacking into private servers is unconstitutional. (Private servers ko hack karna qanoon ke khilaf hai.)
11. **Constitutionally:** The project is constitutionally approved. (Project qanooni taur par approve hai.)

### 5. MIXED ACADEMIC SENTENCES

1.  Algorithms **constitute** the heart of search engines. (Algorithms search engines ka dil hote hain.)
2.  The **constitution** of the lab prevents unauthorized entry. (Lab ka qanoon unauthorized entry ko rokata hai.)
3.  Each **constituent** of the code must be tested. (Code ke har hissay ko test karna chahiye.)
4.  The team is **constituted** by top-level engineers. (Team top-level engineers se bani hai.)
5.  This action is clearly **unconstitutional**. (Ye kaam wazeh taur par qanoon ke khilaf hai.)
6.  Different **constituencies** have different technical needs. (Mukhtalif areas ki technical zarooriat mukhtalif hoti hain.)
7.  **Constituting** a strong password is vital for security. (Strong password banana security ke liye zaroori hai.)
8.  The **constitutive** elements of the CPU are complex. (CPU ke bunyadi hissay bohat complex hain.)
9.  The proposal was **constitutionally** sound. (Proposal qanooni taur par sahi tha.)
10. Many **constituents** make up a database. (Bohat sare hissay mil kar database banate hain.)
11. The new **constitutions** of the club were updated. (Club ke naye qawaneen update kiye gaye.)
12. These **constituencies** are mapped to server locations. (Ye areas server locations ke sath map kiye gaye hain.)
13. Is it **constitutional** to collect user data? (Kya user data collect karna qanoon ke mutabiq hai?)
14. The software **constitutes** a breakthrough in AI. (Ye software AI mein ek badi kamyabi hai.)
15. He represents his **constituency** in the IT board. (Wo IT board mein apne area ko represent karta hai.)
