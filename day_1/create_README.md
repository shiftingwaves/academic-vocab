Hello! I am so happy to help you learn these important academic words. Today, we will learn about the word family **"Create"**. These words are used a lot in Computer Science, like when we "create" a new program or "create" a database.

Let’s start!

---

### 1. FULL WORD FAMILY

*   **Verb forms:** create, creates, created, creating, recreate, recreates, recreated, recreating
*   **Noun forms:** creation, creations, creator, creators
*   **Adjective forms:** creative
*   **Adverb forms:** creatively
*   **Other/Related:** creativity (Noun - the quality of being creative)

---

### 2. EXPLANATION

*   **Create (Verb):** To make something new. (Koi nayi cheez banana.)
*   **Creates (Verb):** He/She makes something. (Woh banata hai.)
*   **Created (Verb):** Made something in the past. (Banaya tha.)
*   **Creating (Verb):** Making something right now. (Bana rahe hain.)
*   **Recreate (Verb):** To make again. (Dobara banana.)
*   **Recreates (Verb):** Makes again (present). (Dobara banata hai.)
*   **Recreated (Verb):** Made again (past). (Dobara banaya.)
*   **Recreating (Verb):** Making again (present continuous). (Dobara bana rahe hain.)
*   **Creation (Noun):** The thing that is made. (Jo cheez bani ho.)
*   **Creations (Noun):** Many things that are made. (Bohat si bani hui cheezein.)
*   **Creator (Noun):** The person who makes it. (Banane wala.)
*   **Creators (Noun):** Many people who make things. (Banane wale log.)
*   **Creative (Adjective):** Having good new ideas. (Naye ideas rakhne wala.)
*   **Creatively (Adverb):** Doing something in a new/smart way. (Naye aur achay tareeqay se.)
*   **Creativity (Noun):** The power to think of new ideas. (Naye ideas sochne ki salahiyat.)

---

### 3. SYNONYMS

*   **Create:** 
    1. Produce (Paida karna ya banana)
    2. Generate (Generate karna / banana)
    3. Construct (Tamir karna / banana)

---

### 4. ACADEMIC EXAMPLES

*   **Create:** The student will create a new website. (Student ek nayi website banaye ga.)
*   **Create:** We need to create a secure password. (Hamein ek safe password banana chahiye.)
*   **Creates:** This software creates a report automatically. (Yeh software khud-ba-khud report banata hai.)
*   **Creates:** Coding creates new possibilities for students. (Coding students ke liye naye raastay banati hai.)
*   **Created:** She created a complex algorithm. (Usne ek mushkil algorithm banaya.)
*   **Created:** The team created a new database. (Team ne ek naya database banaya.)
*   **Creating:** They are creating a mobile app. (Woh ek mobile app bana rahe hain.)
*   **Creating:** Creating clean code is very important. (Saaf code banana bohat zaroori hai.)
*   **Recreate:** We must recreate the experiment to check results. (Hamein results check karne ke liye experiment dobara karna hoga.)
*   **Recreate:** It is hard to recreate that error. (Us error ko dobara banana mushkil hai.)
*   **Creation:** The creation of the internet changed the world. (Internet ki ijaad/banawat ne duniya badal di.)
*   **Creator:** The creator of Python is Guido van Rossum. (Python banane wala Guido van Rossum hai.)
*   **Creative:** He is a very creative programmer. (Woh bohat creative programmer hai.)
*   **Creatively:** We solved the problem creatively. (Humne maslay ko naye tareeqay se hal kiya.)

### 5. MIXED ACADEMIC SENTENCES

1. The researcher is creating a new model for AI. (Researcher AI ke liye naya model bana raha hai.)
2. Many creators share their code on GitHub. (Bohat se banane wale apna code GitHub par share karte hain.)
3. The creation of this software took six months. (Is software ko banane mein 6 mahine lage.)
4. Being creative helps in solving coding bugs. (Creative hona coding bugs hal karne mein madad deta hai.)
5. Developers creatively used the library to finish the project. (Developers ne project khatam karne ke liye library ko naye tareeqay se use kiya.)
6. We must recreate the environment to test the bug. (Hamein bug test karne ke liye environment dobara banana hoga.)
7. The student created a beautiful user interface. (Student ne ek khoobsurat user interface banaya.)
8. Creativity is essential for good software design. (Acche software design ke liye creativity zaroori hai.)
9. The team creates weekly updates for the project. (Team project ke liye har hafte update banati hai.)
10. Who was the creator of this programming language? (Is programming language ko banane wala kaun tha?)
11. We are recreating the database structure. (Hum database structure dobara bana rahe hain.)
12. This tool creates logs of all user activities. (Yeh tool tamam user activities ke logs banata hai.)
13. His creative approach saved a lot of time. (Uske creative tareeqay ne bohat waqt bachaya.)
14. The creations of these students are very impressive. (In students ki banai hui cheezein bohat impressive hain.)
15. She is recreating the graph based on new data. (Woh naye data ke mutabiq graph dobara bana rahi hai.)

| :--- | :--- | :--- |
| **Create** | Banana | Starting a new project/file. |
| **Creation** | Banawat | The result of a process. |
| **Creator** | Banane wala | The author or developer. |
| **Creative** | Naye ideas wala | Smart/Unique design. |

**Memory Trick:** Think of **"Create"** as **"C-R-E-A"** (Computer Related Engineering Activity). Whenever you "create" code, you are doing an Engineering Activity!
