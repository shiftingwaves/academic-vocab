Hello! I am your English instructor. Today, we are going to learn about the word **"Analyse"** (or **"Analyze"**). This is a very important word in Computer Science and academic research.

*Note: "Analyse" is British English, and "Analyze" is American English. Both are correct!*

---

### 1. FULL WORD FAMILY

*   **Verb forms:** analyse / analyze, analysed / analyzed, analyses / analyzes, analysing / analyzing
*   **Noun forms:** analysis, analyst, analysts, analyser, analysers
*   **Adjective forms:** analytic, analytical
*   **Adverb forms:** analytically

---

### 2. EXPLANATION

*   **Analyse / Analyze (Verb):** To look at something carefully to understand how it works.
    *   *Roman Urdu:* Kisi cheez ko ghor se dekhna taake samajh saken ke wo kaise kaam karti hai.
*   **Analysed / Analyzed (Verb - Past):** Already looked at carefully.
    *   *Roman Urdu:* Pehle se ghor se dekha gaya.
*   **Analyses / Analyzes (Verb - Present):** To look at something (he/she/it).
    *   *Roman Urdu:* Ghor se dekhna (jab baat kisi aur ki ho rahi ho).
*   **Analysing / Analyzing (Verb - Continuous):** Currently looking at something carefully.
    *   *Roman Urdu:* Ghor se dekh rahe hain.
*   **Analysis (Noun):** The process of looking at something carefully.
    *   *Roman Urdu:* Ghor se dekhne ka amal (jaiza).
*   **Analyst / Analysts (Noun):** A person who does the analysis.
    *   *Roman Urdu:* Wo shakhs jo cheezon ka jaiza leta hai.
*   **Analyser / Analysers (Noun):** A tool or software that does the analysis.
    *   *Roman Urdu:* Wo tool ya software jo analysis karta hai.
*   **Analytic / Analytical (Adjective):** Related to the skill of thinking logically.
    *   *Roman Urdu:* Soch samajh kar ya logic se kaam karne wala.
*   **Analytically (Adverb):** In a way that shows logical thinking.
    *   *Roman Urdu:* Logic ke sath ya ghor o fikar ke sath.

### 3. SYNONYMS

### 4. ACADEMIC EXAMPLES

*   **Analyse:** Scientists analyse the data to find errors. (Scientists data ko analyse karte hain taake ghaltiyan dhoond saken.)
*   **Analysis:** The analysis of the code took three hours. (Code ka analysis karne mein teen ghante lage.)
*   **Analyst:** The security analyst found a bug. (Security analyst ne aik bug dhoond nikala.)
*   **Analytical:** She has an analytical mind for programming. (Programming ke liye uska dimagh bohat analytical hai.)
*   **Analytically:** The problem was solved analytically. (Masla logic aur ghor o fikar se hal kiya gaya.)

---

### 5. MIXED ACADEMIC SENTENCES

1. You must **analyse** the system requirements first. (Aap ko pehle system requirements ko analyse karna chahiye.)
2. The **analyst** reviewed the network traffic. (Analyst ne network traffic ka jaiza liya.)
3. We are **analysing** the performance of this algorithm. (Hum is algorithm ki performance ko analyse kar rahe hain.)
4. The **analysis** showed a memory leak. (Analysis ne dikhaya ke memory leak ho rahi hai.)
5. Use an **analyser** to check the packet flow. (Packet flow check karne ke liye analyser ka istemal karein.)
6. His **analytical** approach helped fix the bug. (Uske analytical tareeqe ne bug theek karne mein madad ki.)
7. The researchers **analytically** studied the AI model. (Researchers ne AI model ka analytical tareeqe se mutala kiya.)
8. The software **analyses** user input in real-time. (Software real-time mein user input ko analyse karta hai.)
9. Data **analysts** are in high demand. (Data analysts ki bohat demand hai.)
10. The **analysed** results were published yesterday. (Analysed nataij kal shaya kiye gaye.)
11. She **analysed** the security logs carefully. (Usne security logs ko ghor se analyse kiya.)
12. The **analyser** tool identifies syntax errors. (Analyser tool syntax ki ghaltiyan pehchanta hai.)
13. **Analysing** big data requires powerful computers. (Big data ko analyse karne ke liye taqatwar computers chahiye.)
14. An **analytical** report was submitted to the professor. (Professor ko aik analytical report jama karwai gayi.)
15. The team **analysed** the cyber attack patterns. (Team ne cyber attack ke patterns ko analyse kiya.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | Academic Usage |
| :--- | :--- | :--- |
| **Analyse** | To examine closely | Used for data, code, or systems. |
| **Analysis** | A deep study | Used for research findings. |
| **Analyst** | A person who studies | Used for job roles in CS. |
| **Analytical** | Logical thinking | Used for describing someone's skills. |

**Memory Trick:** Think of **"Ana"** (a person) who **"lyses"** (looks/studies) everything carefully. An **Analyst** is just an **Ana** who is a professional at looking!
