Hello! I am your English instructor. Today, we will learn about the word **"Function."** This is a very important word in Computer Science and academic writing. Let’s learn it step-by-step!

---

### 1. FULL WORD FAMILY

*   **Verb forms:**
    *   Function
    *   Functioned
    *   Functioning
*   **Noun forms:**
    *   Function
    *   Functions (plural)
*   **Adjective forms:**
    *   Functional
*   **Adverb forms:**
    *   Functionally

---

### 2. EXPLANATION

*   **Function (Verb):** To work or perform a task.
    *   *Roman Urdu:* Kisi cheez ka kaam karna ya chalna.
*   **Functioned (Verb - Past):** Worked in the past.
    *   *Roman Urdu:* Pehle kaam kiya ya chala.
*   **Functioning (Verb - Continuous):** Working right now.
    *   *Roman Urdu:* Is waqt kaam kar raha hai.
*   **Function (Noun):** A specific purpose or a block of code in programming.
    *   *Roman Urdu:* Kisi cheez ka maqsad ya computer programming mein koi khaas kaam karne wala code.
*   **Functions (Noun - Plural):** More than one purpose or code block.
    *   *Roman Urdu:* Aik se zyada maqsad ya programming functions.
*   **Functional (Adjective):** Something that is working well.
    *   *Roman Urdu:* Woh cheez jo theek se kaam kar rahi ho.
*   **Functionally (Adverb):** In a way that relates to how something works.
    *   *Roman Urdu:* Kaam karne ke hawalay se.

---

### 3. SYNONYMS

*   **Function (Verb):**
    1. **Operate:** Chalna ya kaam karna.
    2. **Work:** Kaam karna.
*   **Function (Noun):**
    1. **Purpose:** Maqsad.
    2. **Role:** Kirdar ya kaam.

---

### 4. ACADEMIC EXAMPLES

*   **Function (V):** The new software must function properly on all devices. (Naye software ko tamam devices par sahi kaam karna chahiye.)
*   **Function (V):** Does this algorithm function as expected? (Kya yeh algorithm umeed ke mutabiq kaam karta hai?)
*   **Functioned (V):** The server functioned well during the test. (Test ke dauran server ne achi tarah kaam kiya.)
*   **Functioned (V):** The old system functioned for ten years. (Purane system ne das saal tak kaam kiya.)
*   **Functioning (V):** The network is functioning slowly today. (Network aaj aahista kaam kar raha hai.)
*   **Functioning (V):** We are functioning under a new security policy. (Hum aik nayi security policy ke tehat kaam kar rahe hain.)
*   **Function (N):** The main function of this code is to store data. (Is code ka asal maqsad data store karna hai.)
*   **Function (N):** Every function in the script must be documented. (Script ke har function ko likha/document kiya jana chahiye.)
*   **Functions (N):** Different functions help the computer process information. (Mukhtalif functions computer ko information process karne mein madad dete hain.)
*   **Functions (N):** This app has many useful functions for students. (Is app mein talaba ke liye bohat se mufeed functions hain.)
*   **Functional (Adj):** The prototype is now fully functional. (Prototype ab mukammal taur par kaam kar raha hai.)
*   **Functional (Adj):** We need a functional design for the website. (Hamein website ke liye aik kaam karne wala design chahiye.)
*   **Functionally (Adv):** The two systems are functionally the same. (Dono systems kaam karne ke lihaz se aik jaise hain.)
*   **Functionally (Adv):** The software is functionally complete. (Software kaam karne ke lihaz se mukammal hai.)

### 5. MIXED ACADEMIC SENTENCES

1. The primary **function** of the CPU is to process instructions. (CPU ka bunyadi kaam instructions ko process karna hai.)
2. Is the keyboard **functioning** correctly? (Kya keyboard sahi tarah kaam kar raha hai?)
3. The new update makes the app more **functional**. (Nayi update app ko zyada kaam ke qabil banati hai.)
4. These **functions** are written in Python. (Yeh functions Python mein likhe gaye hain.)
5. The device **functioned** perfectly during the experiment. (Experiment ke dauran device ne behtareen kaam kiya.)
6. **Functionally**, these two programs are different. (Kaam karne ke lihaz se, yeh dono programs mukhtalif hain.)
7. Can you explain the **function** of this button? (Kya aap is button ka maqsad bata sakte hain?)
8. The database is **functioning** as planned. (Database mansoobay ke mutabiq kaam kar raha hai.)
9. We need to create more **functions** for better performance. (Behtar performance ke liye hamein mazeed functions banane honge.)
10. The system is no longer **functional**. (System ab kaam nahi kar raha.)
11. He **functioned** as the lead developer on this project. (Usne is project par lead developer ke taur par kaam kiya.)
12. The script contains several **functions** for image processing. (Script mein image processing ke liye kayi functions hain.)
13. The design is **functionally** sound. (Yeh design kaam karne ke lihaz se mazboot hai.)
14. How does this system **function** in offline mode? (Yeh system offline mode mein kaise kaam karta hai?)
15. Many **functions** were tested before the launch. (Launch se pehle bohat se functions test kiye gaye.)
