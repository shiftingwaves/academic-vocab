### 2. EXPLANATION

*   **Formula (Noun):** A set of rules or numbers used to solve a problem. (Ek aisa rule ya calculation jo masla hal karne ke liye use ho.)
*   **Formulate (Verb):** To create or prepare a plan or a rule. (Koi plan ya rule banana.)
*   **Formulating (Verb):** Creating a plan right now. (Abhi plan bana rahe hain.)
*   **Formulated (Verb):** Created a plan in the past. (Plan bana liya.)
*   **Formulation (Noun):** The act of creating a plan. (Plan banane ka amal.)
*   **Reformulate (Verb):** To change or make a plan again. (Dobara se plan banana ya theek karna.)
*   **Reformulating (Verb):** Changing a plan again. (Dobara change kar rahe hain.)
*   **Reformulated (Verb):** Changed a plan again. (Dobara change kar liya.)

### 4. ACADEMIC EXAMPLES

*   **Formula:** The computer uses a special **formula** to calculate data. (Computer data calculate karne ke liye ek khaas formula use karta hai.)
*   **Formulae:** Scientists often use complex **formulae** in physics. (Scientists aksar physics mein mushkil formulae use karte hain.)
*   **Formulate:** We need to **formulate** a new strategy for the project. (Hamein project ke liye nayi strategy banani hogi.)
*   **Formulated:** The team **formulated** a plan to fix the software bug. (Team ne software bug theek karne ke liye plan banaya.)
*   **Formulating:** She is **formulating** a hypothesis for her research paper. (Woh apne research paper ke liye hypothesis bana rahi hai.)
*   **Formulation:** The **formulation** of the new algorithm took weeks. (Naye algorithm ko banane ka amal hafton tak chala.)
*   **Reformulate:** You should **reformulate** your argument to be clearer. (Aapko apne argument ko dobara likhna chahiye taake woh saaf ho.)
*   **Reformulated:** The professor **reformulated** the question for the students. (Professor ne students ke liye sawal ko dobara change kiya.)

### 6. CSAVL REVISION SHEET

**Core Meaning:** Creating a rule, plan, or calculation.

| Word | Simple Meaning | Academic Usage |
| :--- | :--- | :--- |
| **Formula** | Rule or Math logic | Used in coding and math. |
| **Formulate** | To create/make | Used to make plans/theories. |
| **Formulation** | The act of making | Used to describe the process. |
| **Reformulate** | To change/fix | Used to improve an existing plan. |

**Memory Trick:**
*   **Form:** Think of "forming" something (banana).
*   **Formula:** A "form" with a "rule" (Ula).
*   **Re-formulate:** "Re" means "again". So, "Make again".
