### 2. EXPLANATION

*   **Role (Noun):**
    *   **English:** The part or function someone or something has in a situation.
    *   **Roman Urdu:** Kisi cheez ya insaan ka kaam ya zimmedari (responsibility). Jaise aik team mein har kisi ka apna role hota hai.
*   **Roles (Noun - Plural):**
    *   **English:** More than one role.
    *   **Roman Urdu:** Aik se zyada kaam ya zimmedariyan.
*   **Role-based (Adjective):**
    *   **English:** Something organized according to the roles people have.
    *   **Roman Urdu:** Jo logon ki zimmedariyon ke hisaab se banaya gaya ho.

---

### 3. SYNONYMS

*   **Function:** Kaam ya maqsad (purpose).
*   **Capacity:** Salahiyat ya position (position).
*   **Duty:** Farz ya zimmedari (responsibility).

---

### 4. ACADEMIC EXAMPLES

*   **Role:**
    1. The computer plays a key **role** in data analysis. (Computer data analysis mein aik ahem kirdar ada karta hai.)
    2. Every student has a specific **role** in the research project. (Har student ka research project mein aik khaas kaam hai.)
*   **Roles:**
    1. Different users have different **roles** in this database system. (Is database system mein mukhtalif users ke mukhtalif kaam hain.)
    2. The software defines the **roles** of all team members. (Software team members ke kirdaron (roles) ko define karta hai.)
*   **Role-based:**
    1. We use a **role-based** security system for this network. (Hum is network ke liye role-based security system use karte hain.)
    2. This is a **role-based** access control model. (Yeh aik role-based access control model hai.)

### 5. MIXED ACADEMIC SENTENCES

1. The **role** of AI in medicine is growing fast. (Medicine mein AI ka kirdar tezi se barh raha hai.)
2. Developers must define **roles** before writing code. (Code likhne se pehle developers ko roles define karne chahiye.)
3. A **role-based** system keeps data safe. (Role-based system data ko mehfooz rakhta hai.)
4. What is the **role** of the CPU in this process? (Is process mein CPU ka kya kaam hai?)
5. We assigned new **roles** to the team members. (Humne team members ko naye kaam diye hain.)
6. **Role-based** permissions help control user access. (Role-based permissions user ki access ko control karne mein madad karti hain.)
7. The **role** of the internet is vital for research. (Research ke liye internet ka kirdar bohat zaroori hai.)
8. Different **roles** require different levels of access. (Mukhtalif kirdaron ko mukhtalif level ki access chahiye hoti hai.)
9. Is this a **role-based** authentication method? (Kya yeh aik role-based authentication method hai?)
10. The main **role** of the server is to store files. (Server ka asal kaam files ko store karna hai.)
11. Many **roles** in IT require coding skills. (IT mein bohat se kirdaron ke liye coding skills chahiye hoti hain.)
12. This **role-based** approach is very efficient. (Yeh role-based tareeqa bohat behtar hai.)
13. Every node has a specific **role** in the network. (Network mein har node ka aik khaas kaam hota hai.)
14. Clear **roles** prevent confusion in large projects. (Wazeh zimmedariyan bade projects mein uljhan se bachati hain.)
15. The **role-based** design is easy to maintain. (Role-based design ko maintain karna aasaan hai.)
