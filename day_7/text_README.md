### 1. FULL WORD FAMILY

*   **Verb forms**
    *   Text (to send a message)
*   **Noun forms**
    *   Text (the main writing)
    *   Texts (plural of text)
    *   Textbook (a book used for study)
*   **Adjective forms**
    *   Textual (related to text)
*   **Adverb forms**
    *   Textually (in a way related to text)

---

### 2. EXPLANATION

*   **Text (Verb):** To send a written message using a phone or computer.
    *   *Roman Urdu:* Phone ya computer ke zariye message bhejna.
*   **Text (Noun):** The written words in a book, file, or article.
    *   *Roman Urdu:* Likhi hui cheez ya alfaaz (jo kisi book ya file mein hon).
*   **Texts (Noun - Plural):** More than one piece of writing.
    *   *Roman Urdu:* Ek se zyada likhi hui cheezein ya documents.
*   **Textbook (Noun):** A book used to learn a subject.
    *   *Roman Urdu:* Woh kitab jo parhai ke liye istemal hoti hai.
*   **Textual (Adjective):** Related to the words or text of something.
    *   *Roman Urdu:* Jo likhe hue alfaaz se talluq rakhta ho.
*   **Textually (Adverb):** In a way that involves text.
    *   *Roman Urdu:* Likhe hue alfaaz ke zariye se.

---

### 4. ACADEMIC EXAMPLES

*   **Text (Verb):**
    1. Please text the results to the research team. (Kripya research team ko results text kar dein.)
    2. The system will text the user when the download finishes. (Jab download khatam hoga, system user ko text karega.)
*   **Text (Noun):**
    1. The text of the paper was very clear. (Research paper ka text bohot saaf tha.)
    2. Computer programs process text data every second. (Computer programs har second text data process karte hain.)
*   **Texts (Noun):**
    1. We analyzed several texts to find the answer. (Humne jawab dhundne ke liye kayi texts ka jaiza liya.)
    2. These texts explain the basics of coding. (Yeh texts coding ke basics samjhate hain.)
*   **Textbook (Noun):**
    1. This textbook is excellent for Java programming. (Yeh textbook Java programming ke liye bohot achi hai.)
    2. The professor recommended a new textbook. (Professor ne ek nayi textbook recommend ki.)
*   **Textual (Adjective):**
    1. The report contains many textual errors. (Report mein bohot si likhai ki galtiyan hain.)
    2. We need to do a textual analysis of the code. (Humein code ka textual analysis karne ki zaroorat hai.)
*   **Textually (Adverb):**
    1. The data is textually represented in the file. (Data file mein textually (likh kar) show kiya gaya hai.)
    2. The changes were textually updated in the system. (System mein tabdeeliyaan textually update kar di gayi thin.)

### 5. MIXED ACADEMIC SENTENCES

1. The researcher read the **text** carefully. (Researcher ne text ko ghaur se parha.)
2. Please **text** the update to your supervisor. (Apne supervisor ko update text kar dein.)
3. Many **texts** are available in the digital library. (Digital library mein bohot saari texts available hain.)
4. The **textbook** covers advanced algorithms. (Textbook mein advanced algorithms cover kiye gaye hain.)
5. There is a **textual** difference between the two files. (Dono files ke darmiyan textual farq hai.)
6. The information is **textually** provided in the manual. (Maloomat manual mein textually di gayi hai.)
7. Coding involves writing a lot of **text**. (Coding mein bohot saara text likhna shamil hai.)
8. The student studied the **textbook** for the exam. (Student ne exam ke liye textbook parhi.)
9. Can you **text** me the link to the paper? (Kya aap mujhe paper ka link text kar sakte hain?)
10. The **textual** content of the website is updated daily. (Website ka textual content rozana update hota hai.)
11. He compared the **texts** to find the errors. (Usne galtiyan dhundne ke liye texts ka mawazna kiya.)
12. The system failed to parse the **text**. (System text ko parse karne mein nakaam raha.)
13. The **textbook** is a great resource for students. (Textbook students ke liye ek behtareen resource hai.)
14. Please ensure the **textual** format is correct. (Baraye meharbani yaqeeni banayein ke textual format sahi ho.)
15. The app **textually** displays the error message. (App error message ko textually dikhati hai.)
