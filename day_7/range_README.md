### 2. EXPLANATION

*   **Range (Verb):** To vary between two points.
    *   *Roman Urdu:* Kisi cheez ka do points ke darmiyan hona ya badalte rehna.
*   **Range (Noun):** The distance or area between two limits.
    *   *Roman Urdu:* Limit ya hadd, jahan tak koi cheez phaili hui ho.
*   **Ranges (Noun/Verb):** Plural form of range.
    *   *Roman Urdu:* Jab ek se zyada limits ya hadd ki baat ho.
*   **Ranged (Verb - Past):** Something that varied in the past.
    *   *Roman Urdu:* Jo guzre hue waqt mein limit ke andar raha ho.
*   **Ranging (Verb/Adjective):** Happening now or describing variety.
    *   *Roman Urdu:* Jab cheezein mukhtalif hadd mein phaili hui hon.

---

### 3. SYNONYMS

1.  **Vary:** (Badalte rehna)
2.  **Scope:** (Dayra-e-kaar / limit)
3.  **Spectrum:** (Ek puri line ya hadd)

---

### 4. ACADEMIC EXAMPLES

*   **Range (Verb):** The test scores range from 50 to 100.
    *   *Roman Urdu:* Test ke numbers 50 se 100 ke darmiyan hain.
*   **Range (Noun):** This software has a wide range of features.
    *   *Roman Urdu:* Is software mein bohat saare (wide range) features hain.
*   **Ranges (Noun):** Different temperature ranges affect CPU performance.
    *   *Roman Urdu:* Mukhtalif temperature ki hadd (ranges) CPU ki speed par asar daalti hain.
*   **Ranged (Verb):** The project costs ranged from $100 to $500.
    *   *Roman Urdu:* Project ka kharcha $100 se $500 ke darmiyan raha.
*   **Ranging (Adjective):** We did a wide-ranging study on AI.
    *   *Roman Urdu:* Hum ne AI par ek bohat barri (wide-ranging) research ki.

### 5. MIXED ACADEMIC SENTENCES

1. The network signal ranges across the entire office. (Network signal pure office mein phaila hua hai.)
2. Our research covers a wide range of topics. (Hamari research bohat saare topics ko cover karti hai.)
3. The server temperatures ranged between 30 and 40 degrees. (Server ka temperature 30 aur 40 degree ke darmiyan raha.)
4. These ranges are used to define data types. (Ye ranges data types ko define karne ke liye use hoti hain.)
5. A ranging search for bugs was conducted. (Bugs dhoondne ke liye ek bari search ki gayi.)
6. The values range from zero to infinity. (Values zero se infinity tak jati hain.)
7. We need to increase the range of our Wi-Fi. (Hamein apne Wi-Fi ki range barhani hogi.)
8. The project involved a wide-ranging analysis of code. (Project mein code ka gehra aur wasee analysis shamil tha.)
9. Data input ranges are set by the programmer. (Data input ki hadd programmer set karta hai.)
10. The system performance ranged from poor to excellent. (System ki performance kharab se behtareen tak rahi.)
11. Different sensors have different ranges of detection. (Mukhtalif sensors ki detection ki hadd alag hoti hai.)
12. The team is currently ranging through the database. (Team abhi database mein cheezein dhoond rahi hai.)
13. Security protocols range in complexity. (Security protocols ki mushkilat mukhtalif hoti hain.)
14. The software supports a full range of file formats. (Software har qisam ke file formats ko support karta hai.)
15. Errors ranged from simple typos to logic faults. (Ghaltiyan chote spelling ki ghaltiyon se lekar logic ki ghaltiyon tak thin.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | CS / Academic Usage |
| :--- | :--- | :--- |
| **Range** | Had ya limit | Used for data limits or signal distance. |

**Memory Trick:** Think of a "Range" as a **"Boundary"** or a **"Fasla"**. If you are playing a game, the "Range" is how far your character can hit!
