Hello! I am your English language instructor. Today, we will learn about the word **"Primary"** and **"Primarily"**. These are very important words in Computer Science and research. Let’s start!

---

### 1. FULL WORD FAMILY

*   **Verb forms:**
    *   None (This word family does not have a common verb form).
*   **Noun forms:**
    *   Primary (as in "a primary") (less common)
*   **Adjective forms:**
    *   Primary
*   **Adverb forms:**
    *   Primarily

---

### 2. EXPLANATION

*   **Primary (Adjective):**
    *   **English:** Most important or first in order.
    *   **Roman Urdu:** Sab se pehla ya sab se ahem (main/important).
*   **Primarily (Adverb):**
    *   **English:** Mainly or mostly.
    *   **Roman Urdu:** Zyada tar, bunyadi tor par.

---

### 3. SYNONYMS

*   **Primary:**
    1.  **Main:** Sab se bada ya ahem.
    2.  **Chief:** Sab se upar ya khaas.
    3.  **Fundamental:** Bunyadi (basic).
*   **Primarily:**
    1.  **Mainly:** Zyada tar.
    2.  **Mostly:** Aksar.
    3.  **Essentially:** Asal mein.

---

### 4. ACADEMIC EXAMPLES

**Primary (Adjective):**
1. The primary goal of this software is to protect user data.
   (Is software ka bunyadi maqsad user ka data mehfooz rakhna hai.)
2. Security is the primary concern in cloud computing.
   (Cloud computing mein security sab se ahem masla/fikr hai.)

**Primarily (Adverb):**
1. This research is primarily based on machine learning algorithms.
   (Ye research zyada tar machine learning algorithms par mabni hai.)
2. Computers are primarily used for processing large amounts of information.
   (Computers ka istemal zyada tar bari miqdar mein information process karne ke liye hota hai.)

---

### 5. MIXED ACADEMIC SENTENCES

1. The primary function of a CPU is to execute instructions.
   (CPU ka bunyadi kaam instructions ko execute karna hai.)
2. This paper is primarily focused on network security.
   (Ye paper zyada tar network security par focus karta hai.)
3. The primary researcher collected data from many sources.
   (Bunyadi researcher ne bohat se sources se data jama kiya.)
4. Our system is primarily designed for mobile users.
   (Hamara system zyada tar mobile users ke liye banaya gaya hai.)
5. Identifying the primary error is the first step in debugging.
   (Bunyadi ghalti ki nishandahi karna debugging ka pehla step hai.)
6. We are primarily using Python for this project.
   (Hum is project ke liye zyada tar Python istemal kar rahe hain.)
7. The primary server is down for maintenance.
   (Bunyadi server maintenance ke liye band hai.)
8. This study is primarily aimed at improving AI speed.
   (Ye study zyada tar AI ki speed behtar karne ke liye hai.)
9. Speed is the primary factor in this hardware design.
   (Is hardware design mein speed sab se ahem factor hai.)
10. The data is primarily stored in a cloud database.
    (Data zyada tar cloud database mein store hota hai.)
11. The primary challenge is to reduce latency.
    (Bunyadi challenge latency ko kam karna hai.)
12. This software is primarily for academic use.
    (Ye software zyada tar academic istemal ke liye hai.)
13. The primary source of power is the battery.
    (Power ka bunyadi source battery hai.)
14. They are primarily looking for new developers.
    (Wo zyada tar naye developers ki talash mein hain.)
15. The primary objective is to finish the code by Friday.
    (Bunyadi maqsad Friday tak code khatam karna hai.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | Common CS Usage |
| :--- | :--- | :--- |
| **Primary** | First / Most important | Primary key, primary goal, primary server |
| **Primarily** | Mainly / Mostly | Primarily used, primarily focused |

**Memory Trick:**
*   Think of **"Primary"** as **"Prime"** (like Prime Minister - the most important person).
*   **Primarily** ends in **"-ly"**, so it describes *how* something is done (mostly).
