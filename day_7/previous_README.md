Hello! I am your English language instructor. Today, we will learn about the words **"previous"** and **"previously"**. These are very important words in academic writing and computer science.

---

### 1. FULL WORD FAMILY

*   **Verb forms:** None (This word does not have a verb form).
*   **Noun forms:**
    *   Previousness (less common)
*   **Adjective forms:**
    *   Previous
*   **Adverb forms:**
    *   Previously

---

### 2. EXPLANATION

*   **Previous (Adjective):**
    *   **Simple English:** Happening or coming before the current time.
    *   **Roman Urdu:** Jo pehle ho chuka ho ya current cheez se zara pehle aaye.
*   **Previously (Adverb):**
    *   **Simple English:** Before now; at an earlier time.
    *   **Roman Urdu:** Is waqt se pehle; pehle kabhi.

---

### 3. SYNONYMS

**For Previous:**
1.  **Prior:** Pehle wala.
2.  **Former:** Guzra hua ya pehle wala.
3.  **Preceding:** Jo kisi cheez se bilkul pehle aaye.

**For Previously:**
1.  **Formerly:** Pehle.
2.  **Earlier:** Is se pehle.

---

### 4. ACADEMIC EXAMPLES

**Previous (Adjective):**
1.  The previous version of the software had many bugs.
    *   (Software ke pichle version mein bohat saare bugs thay.)
2.  Please check your previous research notes before starting the experiment.
    *   (Experiment shuru karne se pehle apne pichle research notes check kar lein.)

**Previously (Adverb):**
1.  The data was previously stored in a local database.
    *   (Data pehle local database mein store kiya gaya tha.)
2.  We have previously discussed the importance of cybersecurity.
    *   (Hum pehle cybersecurity ki ahmiyat par baat kar chuke hain.)

---

### 5. MIXED ACADEMIC SENTENCES

1. The previous experiment failed due to a power issue. (Pichla experiment power issue ki wajah se fail ho gaya.)
2. We previously assumed that the algorithm was efficient. (Hum ne pehle socha tha ke algorithm efficient hai.)
3. Compare this result with your previous findings. (Is result ko apni pichli findings ke sath compare karein.)
4. The server was previously offline for maintenance. (Server pehle maintenance ke liye offline tha.)
5. Did you save the previous configuration? (Kya aap ne pichli configuration save ki thi?)
6. He previously worked as a software developer. (Woh pehle software developer ke tor par kaam karta tha.)
7. The previous chapter explains basic coding rules. (Pichla chapter basic coding rules samjhata hai.)
8. I have previously read this research paper. (Maine yeh research paper pehle parha hai.)
9. The previous update fixed the security error. (Pichle update ne security error theek kar diya.)
10. This method was previously used in older systems. (Yeh method pehle purane systems mein istemal hota tha.)
11. Check the previous logs to find the error. (Error dhundne ke liye pichle logs check karein.)
12. The team previously focused on Artificial Intelligence. (Team ne pehle Artificial Intelligence par tawajo di thi.)
13. Her previous project was about web design. (Uska pichla project web design ke bare mein tha.)
14. Previously, computers were very large in size. (Pehle, computers size mein bohat bade hote thay.)
15. The previous code was much simpler than this one. (Pichla code is wale se kaafi simple tha.)

---

### 6. CSAVL REVISION SHEET

**Core Meaning:** Anything that happened or existed "before now."

| Word | Simple Meaning | Common CS / Academic Usage |
| :--- | :--- | :--- |
| **Previous** | Pehle wala | Used for versions, experiments, or steps. |
| **Previously** | Pehle | Used to describe past actions in research. |

**Memory Trick:**
*   **Pre-** means "before." Think of a **PRE**paid card—you pay *before* you use it. So, **PRE**vious means *before* now!
