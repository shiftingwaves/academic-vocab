### 2. EXPLANATION

*   **Restrict (Verb):** To limit or keep something under control. (Kisi cheez ko limit mein rakhna ya rokna.)
*   **Restricts (Verb):** Same as restrict, used for singular subjects. (Yeh bhi 'rokna' hai, singular subject ke liye.)
*   **Restricting (Verb):** The act of limiting something. (Kisi cheez ko limit karne ka amal.)
*   **Restricted (Verb/Adj):** Limited or not allowed for everyone. (Limit kiya hua ya jo sab ke liye khula na ho.)
*   **Restriction (Noun):** A rule that limits something. (Ek rule ya pabandi jo kisi cheez ko limit karti hai.)
*   **Restrictions (Noun):** Plural of restriction. (Ek se zyada pabandiyan.)
*   **Restrictive (Adj):** Something that stops freedom or limits growth. (Aisi cheez jo freedom ko rok de ya limit kare.)
*   **Unrestricted (Adj):** Free, no limits. (Bilkul free, koi pabandi nahi.)
*   **Unrestrictive (Adj):** Not limiting; allowing freedom. (Jo limit na kare, freedom dene wala.)
*   **Restrictively (Adv):** In a way that limits. (Is tarah se limit karna ke cheez choti ya mehsoos ho.)

### 3. SYNONYMS

    *   **Limit:** Hadd muqarrar karna.

### 4. ACADEMIC EXAMPLES

*   **Restrict:** The firewall restricts access to the server. (Firewall server tak access ko limit karti hai.)
*   **Restrict:** We must restrict the data size for this experiment. (Hamein is experiment ke liye data size ko limit karna hoga.)
*   **Restricts:** The software restricts the user from changing settings. (Software user ko settings change karne se rokta hai.)
*   **Restricts:** This policy restricts unauthorized entry. (Yeh policy ghair-zaroori entry ko rokhti hai.)
*   **Restricting:** Restricting file permissions improves security. (File permissions ko limit karna security behtar karta hai.)
*   **Restricting:** We are restricting access to the database. (Hum database tak access ko limit kar rahe hain.)
*   **Restricted:** Access to the lab is restricted to students. (Lab tak access sirf students tak mehsood/limited hai.)
*   **Restricted:** The file is in a restricted folder. (Yeh file ek limited/restricted folder mein hai.)
*   **Restriction:** There is a restriction on using mobile phones. (Mobile phone use karne par pabandi hai.)
*   **Restriction:** The restriction applies to all guest users. (Yeh pabandi sab guest users par laagu hoti hai.)
*   **Restrictions:** New security restrictions were added today. (Aaj nayi security pabandiyan shamil ki gayi hain.)
*   **Restrictions:** These restrictions help prevent data leaks. (Yeh pabandiyan data leaks ko rokne mein madad karti hain.)
*   **Restrictive:** The rules are too restrictive for developers. (Yeh rules developers ke liye bohat sakht/restrictive hain.)
*   **Restrictive:** A restrictive policy slows down innovation. (Ek sakht policy innovation ko slow kar deti hai.)
*   **Unrestricted:** The system provides unrestricted internet access. (System unrestricted internet access deta hai.)
*   **Unrestricted:** Users have unrestricted rights to edit files. (Users ke paas files edit karne ke unrestricted rights hain.)
*   **Unrestrictive:** The new layout is unrestrictive for users. (Naya layout users ke liye freedom dene wala hai.)
*   **Unrestrictive:** An unrestrictive environment helps creativity. (Ek aazad mahaul creativity mein madad karta hai.)
*   **Restrictively:** The law was applied restrictively. (Qanoon ko bohat sakhti se laagu kiya gaya.)
*   **Restrictively:** We should not define the scope too restrictively. (Hamein scope ko bohat zyada limit nahi karna chahiye.)

### 5. MIXED ACADEMIC SENTENCES

1. The professor will **restrict** the use of AI tools in the exam. (Professor exam mein AI tools ke use ko limit karega.)
2. Many **restrictions** are in place to keep the network safe. (Network ko safe rakhne ke liye bohat si pabandiyan hain.)
3. **Restricting** access to the code prevents errors. (Code tak access ko limit karna errors ko rokta hai.)
4. The **restricted** area is only for senior researchers. (Yeh restricted ilaaqa sirf senior researchers ke liye hai.)
5. Is the internet access **unrestricted** in this lab? (Kya is lab mein internet access free/unrestricted hai?)
6. He **restricts** his research to machine learning only. (Woh apni research ko sirf machine learning tak limit karta hai.)
7. **Restrictive** copyright laws can hurt academic sharing. (Sakht copyright qanoon academic sharing ko nuksan pohancha sakte hain.)
8. The project has no **restrictions** on the tools used. (Project mein use hone wale tools par koi pabandi nahi hai.)
9. The data was used **restrictively** in the final report. (Report mein data ko bohat limited tareeqay se use kiya gaya.)
10. We need an **unrestrictive** policy for team members. (Hamein team members ke liye aazad policy chahiye.)
11. The firewall **restricts** traffic from unknown IPs. (Firewall unknown IPs se aane wale traffic ko rokhti hai.)
12. **Restricted** files are encrypted for safety. (Restricted files safety ke liye encrypt ki gayi hain.)
13. His **restricting** of the budget caused some problems. (Uska budget ko limit karna kuch masail ki wajah bana.)
14. The new update removes previous **restrictions**. (Naya update purani pabandiyon ko khatam kar deta hai.)
15. This is an **unrestricted** version of the software. (Yeh software ka free/unrestricted version hai.)
