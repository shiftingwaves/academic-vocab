### 1. FULL WORD FAMILY

*   **Verb forms:** vary, varies, varied, varying
*   **Noun forms:** variable, variables, variation, variations, variability, variance, variant, variants
*   **Adjective forms:** variable, varied, invariable
*   **Adverb forms:** invariably, variably

---

### 2. EXPLANATION

*   **Vary (verb):** To change or be different. (Cheezon ka badalna ya alag hona.)
*   **Varies (verb):** Same as vary, but for "he/she/it." (Jab koi cheez badalti hai.)
*   **Varied (verb/adj):** Changed or having many types. (Badli hui ya mukhtalif qisam ki.)
*   **Varying (verb):** The act of changing. (Badalte hue.)
*   **Variable (noun/adj):** A value that can change (in coding) or something that changes. (Coding mein wo value jo badal sake.)
*   **Variables (noun):** Plural of variable. (Ek se zyada variables.)
*   **Variation (noun):** A change or a difference. (Tabdeeli ya farq.)
*   **Variations (noun):** Plural of variation. (Mukhtalif qisam ki tabdeeliyan.)
*   **Variability (noun):** The quality of being able to change. (Badalne ki salahiyat.)
*   **Variance (noun):** The state of being different. (Ikhtilaf ya farq.)
*   **Variant (noun):** A different form of something. (Ek cheez ki thori alag shakal.)
*   **Variants (noun):** Plural of variant. (Alag alag forms.)
*   **Invariable (adj):** Never changing. (Jo kabhi na badle.)
*   **Invariably (adverb):** Always, without change. (Hamesha, bina kisi tabdeeli ke.)
*   **Variably (adverb):** In a way that changes. (Is tarah ke badalta rahe.)

---

### 3. SYNONYMS

*   **Vary:** Differ (alag hona), fluctuate (upar niche hona).
*   **Variable:** Changing (badalne wala), unstable (jo mustaqil na ho).
*   **Variation:** Change (tabdeeli), difference (farq).

### 4. ACADEMIC EXAMPLES

*   **Vary:** Data speeds vary depending on your internet connection. (Internet connection ke hisaab se data speed badalti rehti hai.)
*   **Varies:** The error rate varies in different algorithms. (Alag alag algorithms mein ghalti ki sharah badalti rehti hai.)
*   **Variable:** X is a variable in this equation. (Is equation mein X ek variable hai.)
*   **Variables:** We need to define our variables clearly. (Hamein apne variables ko wazeh taur par define karna hoga.)
*   **Variation:** There is a slight variation in the results. (Nataij mein thora sa farq hai.)
*   **Variations:** We tested many variations of the software. (Hum ne software ki bohat si mukhtalif forms test ki.)
*   **Variability:** High variability makes the system hard to predict. (Zyada tabdeeli ki wajah se system ka andaza lagana mushkil hota hai.)
*   **Variance:** The variance in the data was very low. (Data mein jo farq tha wo bohat kam tha.)
*   **Variant:** This is a new variant of the virus. (Yeh virus ki ek nayi qisam/shakal hai.)
*   **Variants:** The app has several variants for different phones. (App ki alag alag phones ke liye mukhtalif forms hain.)
*   **Invariable:** The laws of physics are invariable. (Physics ke qawaneen kabhi nahi badalte.)
*   **Invariably:** The system invariably crashes after an hour. (System hamesha ek ghante baad crash ho jata hai.)
*   **Varied:** The study used varied methods. (Study mein mukhtalif qisam ke tareeqay istemal hue.)
*   **Varying:** We are using varying levels of security. (Hum security ke mukhtalif levels istemal kar rahe hain.)
*   **Variably:** The performance behaved variably. (Performance ka rawaiya badalta raha.)

### 5. MIXED ACADEMIC SENTENCES

1. The output of the program **varies** every time. (Program ka result har dafa badalta hai.)
2. You must declare all **variables** before using them. (Use karne se pehle tumhein saare variables declare karne honge.)
3. There is a small **variation** in the sensor reading. (Sensor ki reading mein chota sa farq hai.)
4. The **invariable** rule is to save your work. (Work save karna ek aisa rule hai jo kabhi nahi badalta.)
5. The code **invariably** fails when the input is empty. (Jab input khali ho to code hamesha fail ho jata hai.)
6. We found many **variants** of the malware. (Hum ne malware ki bohat si aqsaam dhoondi hain.)
7. The **variability** of the network speed is a problem. (Network speed ka bar bar badalna ek masla hai.)
8. The **varied** results show that the test is difficult. (Mukhtalif nataij dikhate hain ke test mushkil hai.)
9. We are **varying** the parameters to see the effect. (Hum asar dekhne ke liye parameters ko badal rahe hain.)
10. The **variance** between the two models is significant. (Dono models ke darmiyan kafi farq hai.)
11. Each **variant** has a unique ID. (Har variant ki ek alag ID hai.)
12. The system performance **variably** depends on the user. (System ki performance user par mukhtalif tarah se depend karti hai.)
13. These **variations** were expected in the research. (Research mein in tabdeeliyon ki tawaqqa thi.)
14. An **invariable** output is required for this function. (Is function ke liye ek jaisa output chahiye.)
15. We compared the **varied** data sets. (Hum ne mukhtalif qisam ke data sets ka mawazna kiya.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | CS / Academic Usage |
| :--- | :--- | :--- |
| **Vary** | To change | Comparing data results. |
| **Variable** | Something that changes | Coding (e.g., `int x = 5`). |
| **Variation** | A difference | Checking errors in research. |
| **Invariable** | Not changing | Constant rules or constants. |
| **Invariably** | Always | Used to describe bugs or rules. |

**Memory Trick:** Think of **"Vary"** as **"Very-different"**. If things **vary**, they are **very different** from each other!
