### 2. EXPLANATION

*   **Normal (Adj):** Following the standard rule or pattern. (Jo cheez bilkul theek aur standard ho.)
*   **Abnormal (Adj):** Not normal; different from what is expected. (Jo cheez aam na ho, kuch ajeeb ho.)
*   **Normally (Adv):** In a normal way. (Aam taur par.)
*   **Abnormally (Adv):** In a way that is not normal. (Ajeeb tareeqay se.)
*   **Normalise/Normalize (Verb):** To make something standard. (Kisi cheez ko standard ya normal banana.)
*   **Normalisation/Normalization (Noun):** The process of making something normal. (Kisi cheez ko standard banane ka amal.)
*   **Normality (Noun):** The state of being normal. (Halat-e-normal.)

---

### 3. SYNONYMS

*   **Normal:** 
    1. Standard (Jo mukarrar ho)
    2. Typical (Jo aam ho)
*   **Abnormal:**
    1. Unusual (Jo aam na ho)
    2. Irregular (Jo be-qaida ho)
*   **Normalize:**
    1. Standardize (Standard ke mutabiq karna)
    2. Regulate (Nizam mein lana)

*   **Normalize:** We must **normalize** the database tables. (Humhein database tables ko normalize karna hoga.) | Please **normalize** the input values. (Baraye meharbani input values ko normalize karein.)
*   **Normalization:** Database **normalization** reduces data errors. (Database normalization data ki galtiyon ko kam karti hai.) | The **normalization** process is complete. (Normalization ka amal mukammal ho gaya hai.)
