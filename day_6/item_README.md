### 1. FULL WORD FAMILY

    *   Itemise

    *   Itemised (often used as an adjective)

    *   None (not commonly used)

---

### 2. EXPLANATION

*   **Item (Noun):** A single thing in a list or group.
    *   *Roman Urdu:* Kisi list ya group ki aik cheez.
*   **Items (Noun - Plural):** More than one thing.
    *   *Roman Urdu:* Aik se zyada cheezain.
*   **Itemise (Verb):** To make a list of things one by one.
    *   *Roman Urdu:* Cheezon ki tafseeli list banana.
*   **Itemises (Verb - 3rd person singular):** He/she/it makes a list.
    *   *Roman Urdu:* Woh list banata hai.
*   **Itemised (Verb/Adj):** Past tense or a list that is already made.
    *   *Roman Urdu:* List banayi gayi; tafseeli list.
*   **Itemising (Verb - Continuous):** The action of making a list.
    *   *Roman Urdu:* List bana raha/rahi hoon.
*   **Itemisation (Noun):** The process of creating a list.
    *   *Roman Urdu:* List banane ka amal.

---

### 3. SYNONYMS

*   **Item:**
    *   **Entry:** List mein darj cheez.
    *   **Element:** Kisi cheez ka hissa (part).
    *   **Unit:** Aik akai/cheez.
*   **Itemise:**
    *   **List:** Cheezon ko likhna.
    *   **Catalog:** Cheezon ki tarteeb se fehrist banana.
    *   **Detail:** Har cheez ko alag alag bayan karna.

### 4. ACADEMIC EXAMPLES

*   **Item:** The database contains every **item** purchased. (Database mein kharidi gayi har cheez shamil hai.)
*   **Items:** Please select all the **items** from the menu. (Baraye meharbani menu se tamam cheezon ko select karain.)
*   **Itemise:** You must **itemise** the expenses in your report. (Aap ko apni report mein kharchon ki tafseeli list banani chahiye.)
*   **Itemises:** The software automatically **itemises** all user logs. (Software khud-ba-khud tamam user logs ki list banata hai.)
*   **Itemised:** The **itemised** bill shows the cost of each part. (Tafseeli bill har hissay ki qeemat dikhata hai.)
*   **Itemising:** The programmer is **itemising** the data files. (Programmer data files ki list bana raha hai.)
*   **Itemisation:** Careful **itemisation** helps in organizing data. (Dhyan se list banana data ko manage karne mein madad karta hai.)

### 5. MIXED ACADEMIC SENTENCES

1. Each **item** in the array has a unique ID. (Array mein har cheez ki aik alag ID hai.)
2. The system stores many **items** in the cloud. (System cloud mein bohot si cheezain store karta hai.)
3. We need to **itemise** the hardware requirements. (Hamein hardware ki zaroori cheezon ki list banani hai.)
4. The script **itemises** every error found during the test. (Script test ke doran milne wali har ghalti ki list banati hai.)
5. I have **itemised** the budget for this project. (Maine is project ke budget ki tafseeli list bana li hai.)
6. She is currently **itemising** the inventory. (Woh is waqt inventory ki list bana rahi hai.)
7. Proper **itemisation** is essential for good research. (Behtar research ke liye cheezon ko list mein rakhna zaroori hai.)
8. The final report includes an **itemised** list of changes. (Final report mein tabdeelion ki tafseeli list shamil hai.)
9. Can you count the total **items** in the list? (Kya aap list mein kul cheezon ko gin sakte hain?)
10. The AI **itemises** data based on user preferences. (AI user ki pasand ke mutabiq data ki list banata hai.)
11. He was **itemising** the files before uploading them. (Woh upload karne se pehle files ki list bana raha tha.)
12. The **itemisation** of the project tasks is finished. (Project ke kamon ki list banane ka kaam khatam ho gaya hai.)
13. Every **item** must be checked for bugs. (Har cheez ko bugs ke liye check karna zaroori hai.)
14. An **itemised** receipt is required for the audit. (Audit ke liye tafseeli receipt zaroori hai.)
15. Please ensure all **items** are saved in the database. (Baraye meharbani yaqeen banayen ke tamam cheezain database mein save hain.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | CS / Academic Usage |
| :--- | :--- | :--- |
| **Item** | A single thing | Used for array elements or database records. |
| **Itemise** | Make a list | Used when documenting expenses or logs. |
| **Itemisation** | The act of listing | Used in reports to show structure. |

**Memory Trick:** Think of **"Item"** as a single **"I"** (Individual) **"tem"** (thing). If you "Itemise," you are putting "I"s into a list!
