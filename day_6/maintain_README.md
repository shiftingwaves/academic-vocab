### 1. FULL WORD FAMILY

*   **Verb forms:** Maintain, maintained, maintaining, maintains.
*   **Noun forms:** Maintenance.
*   **Adjective forms:** Maintainable (less common).
*   **Adverb forms:** None.

---

### 2. EXPLANATION

*   **Maintain (Verb):** To keep something in good condition or at a certain level.
    *   *Roman Urdu:* Kisi cheez ko sahi halat mein rakhna ya usay jari rakhna.
*   **Maintained (Verb - Past):** Kept in good condition in the past.
    *   *Roman Urdu:* Pehle se sahi halat mein rakha gaya.
*   **Maintaining (Verb - Continuous):** Keeping something going right now.
    *   *Roman Urdu:* Kisi cheez ko jari rakhna ya sambhal kar rakhna.
*   **Maintains (Verb - Present):** Keeps something in good condition (used for he/she/it).
    *   *Roman Urdu:* Woh hamesha sahi halat mein rakhta hai.
*   **Maintenance (Noun):** The process of keeping something in good condition.
    *   *Roman Urdu:* Kisi cheez ki dekh bhaal ya marammat (repair).
*   **Maintainable (Adjective):** Something that is easy to keep in good condition.
    *   *Roman Urdu:* Aisi cheez jisay aasani se sambhala ja sakay.

---

### 3. SYNONYMS

*   **Preserve:** Hifazat karna ya mehfooz rakhna.
*   **Sustain:** Kisi cheez ko jari rakhna (continue).
*   **Keep up:** Barqarar rakhna.

---

### 4. ACADEMIC EXAMPLES

*   **Maintain:** The software needs to maintain high speed. (Software ko apni speed tez rakhni chahiye.)
*   **Maintained:** The server maintained a stable connection. (Server ne connection ko stable rakha.)
*   **Maintaining:** Maintaining code quality is important for developers. (Code ki quality ko behtar rakhna developers ke liye zaroori hai.)
*   **Maintains:** This system maintains data security. (Yeh system data ki hifazat barqarar rakhta hai.)
*   **Maintenance:** Regular maintenance prevents system crashes. (Regular dekh bhaal se system crash hone se bach jata hai.)
*   **Maintainable:** The new code is very maintainable. (Naya code sambhalne mein bahut aasaan hai.)

---

### 5. MIXED ACADEMIC SENTENCES

1. Developers must maintain clean code. (Developers ko saaf code rakhna chahiye.)
2. We maintained the database for three years. (Hum ne teen saal tak database ko sambhala.)
3. He is maintaining the server logs. (Woh server logs ko maintain kar raha hai.)
4. The system maintains user privacy. (System user ki privacy ko barqarar rakhta hai.)
5. Proper maintenance improves hardware life. (Sahi dekh bhaal hardware ki zindagi barhati hai.)
6. Is this code maintainable? (Kya yeh code sambhalne ke qabil hai?)
7. We must maintain high standards in research. (Humein research mein high standards rakhne chahiye.)
8. The lab maintained a steady temperature. (Lab ka temperature barqarar rakha gaya.)
9. She is maintaining a list of errors. (Woh errors ki list maintain kar rahi hai.)
10. The website maintains a large database. (Website ek bara database sambhalti hai.)
11. Maintenance of the network is expensive. (Network ki dekh bhaal mehngi hai.)
12. This structure is not easily maintainable. (Yeh structure aasani se maintain nahi ho sakta.)
13. They maintained contact with the professor. (Unhon ne professor se rabta barqarar rakha.)
14. Maintaining focus is key to success. (Focus barqarar rakhna kamyabi ki kunji hai.)
15. The system maintains its speed under load. (System load ke bawajood apni speed maintain rakhta hai.)
