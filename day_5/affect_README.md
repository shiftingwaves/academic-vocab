### 1. FULL WORD FAMILY

*   **Verb forms:**
    *   Affect
    *   Affects
    *   Affected
    *   Affecting
*   **Noun forms:**
    *   Affect (less common - used in psychology)
*   **Adjective forms:**
    *   Affective (related to feelings/emotions)
    *   Unaffected
*   **Adverb forms:**
    *   Affectively

---

### 2. EXPLANATION

*   **Affect (verb):** To change something or have an impact on it.
    *   *Roman Urdu:* Kisi cheez par asar daalna ya badlaav lana.
*   **Affects (verb):** The action happens to a singular thing (he/she/it).
    *   *Roman Urdu:* Jab koi cheez kisi dusri cheez par asar daalti hai.
*   **Affected (verb/adj):** Changed by something.
    *   *Roman Urdu:* Asar hua, ya badal gaya.
*   **Affecting (verb):** Currently changing something.
    *   *Roman Urdu:* Asar daal raha hai.
*   **Affect (noun):** An emotional feeling.
    *   *Roman Urdu:* Jazbaat ya mood ka izhaar.
*   **Affective (adj):** Related to moods or feelings.
    *   *Roman Urdu:* Jazbaat se talluq rakhne wala.
*   **Unaffected (adj):** Not changed by something.
    *   *Roman Urdu:* Jis par koi asar na hua ho, bilkul theek.
*   **Affectively (adv):** In a way that relates to feelings.
    *   *Roman Urdu:* Jazbaati taur par.

---

### 3. SYNONYMS

*   **Affect (Verb):**
    *   **Influence:** Asar daalna.
    *   **Impact:** Gehra asar hona.
    *   **Alter:** Tabdeel karna ya badalna.

---

### 4. ACADEMIC EXAMPLES

*   **Affect:** Poor internet speed can **affect** your research work.
    *   *Roman Urdu:* Internet ki kharab speed aapke research ke kaam par asar daal sakti hai.
*   **Affects:** A virus often **affects** the system memory.
    *   *Roman Urdu:* Virus aksar system ki memory par asar daalta hai.
*   **Affected:** The server was **affected** by the power failure.
    *   *Roman Urdu:* Power failure ki wajah se server mutasir (affected) hua.
*   **Affecting:** We are studying factors **affecting** AI performance.
    *   *Roman Urdu:* Hum un wajoohaat par parh rahe hain jo AI ki performance par asar daal rahi hain.
*   **Affect (Noun):** The robot showed a neutral **affect**.
    *   *Roman Urdu:* Robot ne koi khaas jazbaat (affect) show nahi kiye.
*   **Affective:** This study focuses on the **affective** state of users.
    *   *Roman Urdu:* Yeh study users ke jazbaati (affective) halaat par focus karti hai.
*   **Unaffected:** The core database remained **unaffected** by the update.
    *   *Roman Urdu:* Update hone ke bawajood core database par koi asar nahi hua.
*   **Affectively:** The AI responded **affectively** to the user's frustration.
    *   *Roman Urdu:* AI ne user ki pareshani ka jazbaati (affectively) jawab diya.

### 5. MIXED ACADEMIC SENTENCES

1.  Low memory **affects** how fast the computer runs. (Kam memory computer ki speed par asar daalti hai.)
2.  The software update has **affected** our system stability. (Software update ne hamare system ki stability ko mutasir kiya hai.)
3.  Are you **affecting** the code while testing? (Kya aap testing ke dauran code ko tabdeel kar rahe hain?)
4.  The study shows that colors can be **affective**. (Study dikhati hai ke rang jazbaati asar daal sakte hain.)
5.  Many files remained **unaffected** after the virus scan. (Virus scan ke baad bahut si files par koi asar nahi hua.)
6.  The researcher analyzed the **affective** response of the participants. (Researcher ne participants ke jazbaati rad-e-amal ka tajziya kiya.)
7.  How does heat **affect** the hardware? (Garmi hardware par kaise asar karti hai?)
8.  The **affect** of the user was clearly visible in the data. (User ke jazbaat data mein saaf nazar aa rahe thay.)
9.  We must understand the variables **affecting** our model. (Humein un variables ko samajhna hoga jo hamare model par asar daal rahe hain.)
10. The system performed **affectively** in a simulation. (System ne simulation mein jazbaati taur par sahi kaam kiya.)
11. Nothing is **unaffected** by the new security policy. (New security policy se koi bhi cheez asar se mehfooz nahi hai.)
12. Does this bug **affect** all users? (Kya yeh bug sab users par asar daalta hai?)
13. The **affected** files were moved to a separate folder. (Mutasir shuda files ko alag folder mein move kar diya gaya.)
14. Please check if the sensor is **unaffected** by light. (Check karein ke sensor roshni se mutasir toh nahi ho raha.)
15. This change will **affect** the final results. (Yeh tabdeeli final results par asar daalegi.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | Academic/CS Usage |
| :--- | :--- | :--- |
| **Affect** | To change | Used to describe how one variable changes another. |
| **Affected** | Changed | Used for systems or files that have been damaged. |
| **Affecting** | Changing | Used for ongoing processes or bugs. |
| **Affective** | Emotional | Used in AI for human-computer interaction (HCI). |
| **Unaffected** | Not changed | Used to show that a system is safe or stable. |

**Memory Trick:**
Think of **A-ffect** as **A-ction**. If you take an *action* on a computer file, you *affect* it!
