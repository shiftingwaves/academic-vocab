Hello! I am your English language instructor. Today, we will learn about the word family **"Consume."** This word is very important in Computer Science, especially when we talk about how much power or memory a computer program uses.

Let's start our lesson!

---

### 1. FULL WORD FAMILY

*   **Verb forms:**
    *   Consume (Base form)
    *   Consumes (Present tense)
    *   Consumed (Past tense)
    *   Consuming (Continuous)
*   **Noun forms:**
    *   Consumer (Person or thing that uses something)
    *   Consumers (Plural)
    *   Consumption (The act of using something)
*   **Adjective forms:**
    *   Consumable (Something that can be used up - *less common*)
*   **Adverb forms:**
    *   None

---

### 2. EXPLANATION

*   **Consume:** To use up resources (like energy, time, or memory).
    *   *Roman Urdu:* Kisi cheez ko istemaal kar lena ya khatam kar dena.
*   **Consumes:** Used for singular subjects (He/She/It).
    *   *Roman Urdu:* Jab koi cheez resource istemaal karti hai.
*   **Consumed:** Something that was used in the past.
    *   *Roman Urdu:* Jo cheez pehle istemaal ho chuki ho.
*   **Consuming:** The process of using something right now.
    *   *Roman Urdu:* Kisi cheez ka istemaal hona (process mein).
*   **Consumer:** A person or a system that uses a product.
    *   *Roman Urdu:* Woh jo cheez istemaal karta hai (user).
*   **Consumers:** Multiple users or systems.
    *   *Roman Urdu:* Bohat saare users.
*   **Consumption:** The act of using something.
    *   *Roman Urdu:* Istemal ka amal.

---

### 3. SYNONYMS

*   **Consume:**
    1.  **Use:** Istemal karna.
    2.  **Utilize:** Kisi cheez ko kaam mein lana.
    3.  **Exhaust:** Poori tarah khatam kar dena.

### 4. ACADEMIC EXAMPLES

*   **Consume:** This software will **consume** a lot of RAM. (Yeh software bohat zyada RAM istemaal karega.)
*   **Consume:** Do not **consume** too much bandwidth. (Bohat zyada bandwidth istemaal mat karo.)
*   **Consumes:** The background process **consumes** 5% of the CPU. (Background process 5% CPU istemaal karta hai.)
*   **Consumes:** My script **consumes** less memory now. (Mera script ab kam memory istemaal karta hai.)
*   **Consumed:** The update **consumed** all the disk space. (Update ne saari disk space khatam kar di.)
*   **Consumed:** We **consumed** the data in the database. (Humne database ka data istemaal kar liya.)
*   **Consuming:** The program is **consuming** too much power. (Program bohat zyada power istemaal kar raha hai.)
*   **Consuming:** He is **consuming** API services daily. (Woh rozana API services istemaal kar raha hai.)
*   **Consumer:** The **consumer** needs a fast internet connection. (User ko fast internet connection ki zaroorat hai.)
*   **Consumer:** Every **consumer** wants a better app. (Har user behtar app chahta hai.)
*   **Consumption:** High power **consumption** is bad for laptops. (Zyada power ka istemaal laptop ke liye bura hai.)
*   **Consumption:** We must lower data **consumption**. (Hamein data ka istemaal kam karna chahiye.)

### 5. MIXED ACADEMIC SENTENCES

1. This algorithm **consumes** very little time. (Yeh algorithm bohat kam waqt leta hai.)
2. Large files **consume** significant storage space. (Bari files bohat zyada storage leti hain.)
3. The **consumer** of this data is a mobile app. (Is data ka user ek mobile app hai.)
4. High CPU **consumption** slows down the server. (Zyada CPU ka istemaal server ko slow kar deta hai.)
5. We have **consumed** all available memory. (Humne saari available memory istemaal kar li hai.)
6. Stop **consuming** resources unnecessarily. (Fazool mein resources istemaal karna band karo.)
7. Many **consumers** prefer cloud storage today. (Aaj kal bohat se users cloud storage pasand karte hain.)
8. The system **consumes** energy even in sleep mode. (System sleep mode mein bhi energy istemaal karta hai.)
9. Excessive power **consumption** leads to overheating. (Zyada power istemaal hone se device garam ho jati hai.)
10. The application **consumed** the API response quickly. (Application ne API response jaldi istemaal kar liya.)
11. Are you **consuming** the data from the cloud? (Kya tum cloud se data istemaal kar rahe ho?)
12. This new processor **consumes** less electricity. (Yeh naya processor kam bijli istemaal karta hai.)
13. The **consumer** expects a fast response. (User ko tez response ki umeed hoti hai.)
14. Reduce the **consumption** of background tasks. (Background tasks ka istemaal kam karo.)
15. We **consumed** the entire dataset for training. (Humne training ke liye poora dataset istemaal kiya.)
