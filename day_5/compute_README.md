---

### 1. FULL WORD FAMILY

*   **Verb forms:** Compute, Computed (past), Computing (present continuous)
*   **Noun forms:** Computer, Computers, Computation, Computations
*   **Adjective forms:** Computable, Computational, Computerised
*   **Adverb forms:** Computationally

---

### 2. EXPLANATION

*   **Compute (Verb):** To calculate or use a machine to solve a math problem. (Math ya calculation solve karna).
*   **Computed (Verb):** Did the calculation in the past. (Calculation kar li).
*   **Computing (Verb/Noun):** The act of using computers or doing math. (Computer istemal karna ya calculation ka kaam).
*   **Computer (Noun):** An electronic machine for data. (Woh machine jo data process karti hai).
*   **Computers (Noun):** More than one computer. (Ek se zyada computers).
*   **Computation (Noun):** A calculation or result of math. (Math ka koi jawab ya calculation).
*   **Computations (Noun):** Many calculations. (Bohat saari calculations).
*   **Computable (Adjective):** Something that can be calculated. (Jo cheez solve ho sakti ho).
*   **Computational (Adjective):** Related to the process of calculating. (Calculation se mutaliq).
*   **Computerised (Adjective):** Something controlled by a computer. (Jo cheez computer se chalti ho).
*   **Computationally (Adverb):** In a way that uses a computer. (Computer ki madad se).

---

### 3. SYNONYMS

*   **Compute:** Calculate (hisaab lagana), Reckon (andaaza lagana).
*   **Computation:** Calculation (hisaab), Analysis (tajziya).
*   **Computerised:** Automated (khud-ba-khud chalne wala), Digital (digital).

---

### 4. ACADEMIC EXAMPLES

*   **Compute:** We need to compute the total data size. (Humein total data ka size calculate karna hai.)
*   **Computed:** The system computed the results quickly. (System ne results jaldi calculate kar liye.)
*   **Computing:** Cloud computing is very popular today. (Aaj kal cloud computing bohat mashhoor hai.)
*   **Computer:** This computer is very fast. (Yeh computer bohat tez hai.)
*   **Computers:** Modern computers use AI. (Jadeed computers AI istemal karte hain.)
*   **Computation:** This computation is very complex. (Yeh calculation bohat mushkil hai.)
*   **Computations:** These computations require high power. (In calculations ke liye zyada power chahiye.)
*   **Computable:** Is this problem computable? (Kya yeh masla solve ho sakta hai?)
*   **Computational:** It is a computational task. (Yeh calculation se juda kaam hai.)
*   **Computerised:** We use a computerised system. (Hum ek computer se chalne wala system use karte hain.)
*   **Computationally:** It is computationally expensive. (Yeh computer ke liye mushkil kaam hai.)

### 5. MIXED ACADEMIC SENTENCES

1. Scientists **compute** the speed of light. (Scientists light ki speed calculate karte hain.)
2. The **computer** crashed during the test. (Test ke doran computer band ho gaya.)
3. **Computers** are essential for research. (Research ke liye computers zaroori hain.)
4. The **computation** took three hours. (Calculation mein teen ghante lage.)
5. Complex **computations** are done by servers. (Mushkil calculations servers karte hain.)
6. The model is **computable** by software. (Yeh model software se solve ho sakta hai.)
7. **Computational** biology is a new field. (Computational biology ek naya field hai.)
8. The bank uses a **computerised** system. (Bank ek computerised system istemal karta hai.)
9. This task is **computationally** difficult. (Yeh kaam computer ke liye mushkil hai.)
10. **Computing** power increases every year. (Computing power har saal barhti hai.)
11. We **computed** the final average score. (Humne final average score nikal liya.)
12. The **computerised** robot works well. (Computer se chalne wala robot achi tarah kaam karta hai.)
13. New **computational** tools are helpful. (Naye calculation wale tools madadgaar hain.)
14. The **computation** was accurate. (Calculation bilkul sahi thi.)
15. Many **computers** are connected in a network. (Bohat se computers ek network mein jude hain.)
