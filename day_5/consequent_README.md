---

### 2. EXPLANATION

*   **Consequence (Noun):** A result or effect of an action.
    *   *Roman Urdu:* Kisi kaam ka natija ya asar.
*   **Consequences (Noun):** More than one result.
    *   *Roman Urdu:* Aik se zyada natijay.
*   **Consequent (Adjective):** Something that happens because of something else.
    *   *Roman Urdu:* Jo kisi cheez ke natijay mein ho.
*   **Consequently (Adverb):** Therefore; as a result.
    *   *Roman Urdu:* Is liye, ya natijatan.

---

### 3. SYNONYMS

*   **Consequence:**
    1. Result (Natija)
    2. Outcome (Anjaam)
*   **Consequently:**
    1. Therefore (Is liye)
    2. Thus (Is tarah / is wajah se)

---

### 4. ACADEMIC EXAMPLES

*   **Consequence:**
    1. Poor code quality is a direct consequence of bad planning. (Ghatiya coding, bad planning ka natija hoti hai.)
    2. The main consequence of this error is system failure. (Is ghalti ka sab se bara natija system ka fail hona hai.)
*   **Consequences:**
    1. You must understand the security consequences of using weak passwords. (Aap ko kamzor passwords use karne ke security natijay samajhne chahiye.)
    2. The project failed due to several negative consequences. (Project kayi manfi nataij ki wajah se fail ho gaya.)
*   **Consequent:**
    1. The system crash and the consequent loss of data were worrying. (System crash aur uske natijay mein data ka khatam hona pareshan kun tha.)
    2. We studied the error and the consequent delay in the project. (Hum ne ghalti aur uske natijay mein hone wali takheer ka mutala kiya.)
*   **Consequently:**
    1. The server was overloaded; consequently, the website became slow. (Server par bojh zyada tha; is liye website slow ho gayi.)
    2. No testing was done; consequently, the software has many bugs. (Testing nahi ki gayi; natijatan software mein bohat se bugs hain.)

### 5. MIXED ACADEMIC SENTENCES

1. The update caused a bug; consequently, the app crashed. (Update ne bug paida kiya; natijatan app crash ho gayi.)
2. A consequence of this algorithm is faster processing. (Is algorithm ka aik natija fast processing hai.)
3. The server failed, and the consequent downtime affected many users. (Server fail ho gaya, aur uske natijay mein hone wali downtime ne users ko mutasir kiya.)
4. He didn't update the firewall; consequently, the system was hacked. (Usne firewall update nahi ki; is liye system hack ho gaya.)
5. We must consider the long-term consequences of using this database. (Hamein is database ko use karne ke lambe arsay ke nataij par ghaur karna chahiye.)
6. The electricity went out; consequently, we lost our unsaved work. (Bijli chali gayi; is liye hum ne apna save na kiya hua kaam kho diya.)
7. The consequent increase in traffic slowed down the network. (Traffic mein hone wale izafay ne network ko slow kar diya.)
8. Lack of documentation is a major consequence of poor management. (Documentation ki kami buri management ka bara natija hai.)
9. The code was messy; consequently, it was hard to debug. (Code ganda tha; is liye use debug karna mushkil tha.)
10. We analyzed the consequences of changing the network settings. (Hum ne network settings badalne ke nataij ka tajziya kiya.)
11. The error was minor, and the consequent fix was simple. (Ghalti choti thi, aur uske natijay mein hone wala fix aasaan tha.)
12. The battery died; consequently, the laptop turned off. (Battery khatam ho gayi; is liye laptop band ho gaya.)
13. One consequence of AI is the change in job markets. (AI ka aik natija job markets mein tabdeeli hai.)
14. The system was outdated; consequently, it could not run the new software. (System purana tha; is liye wo naya software nahi chala saka.)
15. The consequent loss of files was a disaster for the team. (Files ka natijatan khatam hona team ke liye tabahi tha.)

### 6. CSAVL REVISION SHEET

| Word | Simple Meaning | Academic Usage |
| :--- | :--- | :--- |
| **Consequence** | Result | Used to describe what happens after an event. |
| **Consequences** | Results | Used for multiple effects (usually bad). |
| **Consequent** | Happening after | Used to describe a result that follows an event. |
| **Consequently** | Therefore | Used to connect a cause to a result (Formal). |

**Memory Trick:**
Think of **"Consequently"** as a formal version of **"So."**
*   *Casual:* The internet is down, **so** I cannot work.
*   *Academic:* The internet is down; **consequently**, I cannot work.
