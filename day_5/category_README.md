### 1. FULL WORD FAMILY

*   **Verb forms:** categorise (UK), categorize (US), categorised, categorizes, categorizing, categorising.
*   **Noun forms:** category, categories, categorisation (UK), categorization (US).
*   **Adjective forms:** categorical (less common).
*   **Adverb forms:** categorically (less common).

---

### 2. EXPLANATION

*   **Category (Noun):** A group of things that are similar. (Cheezon ka group jo ek jaisi hon).
*   **Categories (Noun, plural):** More than one group. (Ek se zyada groups).
*   **Categorise/Categorize (Verb):** To put things into groups. (Cheezon ko alag alag groups mein rakhna).
*   **Categorised/Categorized (Verb, past):** Put into groups in the past. (Pichle waqt mein groups mein rakha gaya).
*   **Categorises/Categorizes (Verb, present):** To put into groups (for he/she/it). (Groups mein rakhna).
*   **Categorising/Categorizing (Verb, continuous):** The act of putting things into groups. (Groups mein rakhne ka amal).
*   **Categorisation/Categorization (Noun):** The process of grouping. (Grouping karne ka tariqa).
*   **Categorical (Adjective):** Clear and direct. (Bilkul saaf aur wazeh).
*   **Categorically (Adverb):** In a clear and direct way. (Saaf taur par).

### 4. ACADEMIC EXAMPLES

*   **Category:** The software has a new category for security files.
    *(Is software mein security files ke liye ek naya category hai.)*
*   **Categories:** The database uses different categories for user data.
    *(Database user data ke liye mukhtalif categories istemal karta hai.)*
*   **Categorize:** You must categorize the data before analysis.
    *(Analysis se pehle aapko data ko categorize karna hoga.)*
*   **Categorized:** We categorized the errors by their type.
    *(Humne errors ko unki qisam ke mutabiq categorize kiya.)*
*   **Categorization:** Proper categorization helps in searching files quickly.
    *(Sahi categorization files ko jaldi dhoondne mein madad karti hai.)*
