### 1. FULL WORD FAMILY

*   **Verb forms:**
    *   Construct
    *   Constructs
    *   Constructing
    *   Constructed
    *   Reconstruct
    *   Reconstructs
    *   Reconstructing
    *   Reconstructed
*   **Noun forms:**
    *   Construction
    *   Constructions
    *   Reconstruction
*   **Adjective forms:**
    *   Constructive
*   **Adverb forms:**
    *   Constructively (less common)

---

### 2. EXPLANATION

*   **Construct (Verb):** To build or make something. (Kisi cheez ko banana ya tayar karna.)
*   **Constructs (Verb):** Third-person form of construct. (Jab hum kisi ek cheez ya 'he/she/it' ki baat karte hain.)
*   **Constructing (Verb):** The act of building. (Koi cheez bana rahe hona.)
*   **Constructed (Verb):** Made in the past. (Jo cheez pehle ban chuki ho.)
*   **Reconstruct (Verb):** To build again. (Dobara banana.)
*   **Reconstructs (Verb):** To build again (third-person). (Dobara banana.)
*   **Reconstructing (Verb):** Building something again. (Dobara bana rahe hona.)
*   **Reconstructed (Verb):** Built again in the past. (Jo cheez dobara ban chuki ho.)
*   **Construction (Noun):** The process of building. (Banane ka amal.)
*   **Constructions (Noun):** Things that are built. (Banayi gayi cheezein.)
*   **Reconstruction (Noun):** The act of building something again. (Dobara banane ka amal.)
*   **Constructive (Adjective):** Useful or helpful. (Jo cheez faidamand ho aur behtar banane mein madad de.)

---

### 3. SYNONYMS

*   **Construct (Verb):**
    *   **Build:** Banana ya tameer karna.
    *   **Create:** Koi nayi cheez paida karna.
    *   **Assemble:** Parts ko jor kar cheez banana.

### 4. ACADEMIC EXAMPLES

*   **Construct:** Developers construct new software using Python. (Developers Python ka istemal kar ke naya software banate hain.)
*   **Constructs:** The system constructs a data map. (System ek data map banata hai.)
*   **Constructing:** We are constructing a new database model. (Hum ek naya database model bana rahe hain.)
*   **Constructed:** The robot was constructed in the lab. (Robot lab mein banaya gaya tha.)
*   **Reconstruct:** Researchers try to reconstruct lost files. (Researchers khoyi hui files ko dobara banane ki koshish karte hain.)
*   **Reconstructs:** The software reconstructs the corrupted image. (Software kharab image ko dobara theek karta hai.)
*   **Reconstructing:** He is reconstructing the server architecture. (Woh server architecture ko dobara bana raha hai.)
*   **Reconstructed:** The data was reconstructed after the crash. (Crash ke baad data ko dobara banaya gaya.)
*   **Construction:** The construction of the network took two weeks. (Network banane ka amal do hafte mein mukammal hua.)
*   **Constructions:** These complex constructions require high memory. (Ye pecheeda cheezein zyada memory maangti hain.)
*   **Reconstruction:** Reconstruction of the code is necessary. (Code ko dobara banana zaroori hai.)
*   **Constructive:** Please give constructive feedback on my code. (Barah-e-karam mere code par achi aur faidamand raye dein.)
